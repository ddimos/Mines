# Mines
A SFML multiplayer copy of minesweeper

![game3](https://user-images.githubusercontent.com/43299351/221378722-4018794f-da8f-476f-9f66-92c9e3e11cef.png)

## Installation
### Ubuntu 
Install sfml libs
```bash
sudo apt-get install libsfml-dev
```

## Launch
Build
```bash
make
```
Launch the game
```bash
./bin/main
```
