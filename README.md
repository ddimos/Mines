# Mines
A SFML multiplayer copy of minesweeper

![game](https://user-images.githubusercontent.com/43299351/221376906-f8c372a4-b027-4d8b-937e-9a89b3c2b7de.png)

## Installation
### Ubuntu 
Install sfml libs
```bash
sudo apt-get install libsfml-dev
```

## Launch
Build
```bash
make
```
Launch the game
```bash
./bin/main
```
